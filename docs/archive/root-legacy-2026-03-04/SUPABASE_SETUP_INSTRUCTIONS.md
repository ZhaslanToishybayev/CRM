## 🔧 Структура базы данных:

### user_stats
```sql
id (UUID) → auth.users.id
username (TEXT)
email (TEXT)
level (INT) - уровень пользователя
xp (INT) - опыт
streak (INT) - дни подряд
```

### tasks
```sql
id (UUID)
user_id (UUID) → auth.users.id
title (TEXT)
description (TEXT)
category (ENUM) - Work, Personal, Health, Shopping, General
priority (INT) - 1, 2, 3
status (ENUM) - pending, in_progress, completed
due_date (TIMESTAMP)
```

## 🆘 Проблемы?

### Ошибка подключения к Supabase
- Проверьте интернет-соединение
- Проверьте URL и ключ в `supabase_config.dart`

### AI не работает
- Проверьте настройки в `ai_settings_screen.dart`
- Убедитесь что API ключ GLM корректен

### Данные не сохраняются
- Проверьте что таблицы созданы
- Проверьте RLS политики
- Убедитесь что пользователь авторизован
