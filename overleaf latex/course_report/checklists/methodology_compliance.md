# Чек-лист соответствия методическим требованиям

## 1. Структура работы
- [x] Титульный лист
- [x] Содержание
- [x] Введение
- [x] Основная часть: 2 раздела
- [x] Заключение
- [x] Список использованной литературы
- [x] Приложения

## 2. Форматирование
- [x] Формат страницы A4
- [x] Поля: левое 30 мм, верх/низ 20 мм, правое 10 мм
- [x] Основной шрифт Times New Roman (с fallback TeX Gyre Termes)
- [x] Размер шрифта 14 pt
- [x] Межстрочный интервал 1.5
- [x] Абзацный отступ 1.25 см

## 3. Нумерация страниц
- [x] Титульный лист и содержание учитываются в общей нумерации
- [x] Номер страницы на титуле и содержании не выводится
- [x] Отображаемая нумерация начинается со страницы "Введение" (стр. 3)

## 4. Заголовки и разделы
- [x] "ВВЕДЕНИЕ", "ЗАКЛЮЧЕНИЕ", "СПИСОК ИСПОЛЬЗОВАННОЙ ЛИТЕРАТУРЫ" оформлены как самостоятельные заголовки
- [x] Раздел 1 содержит несколько подразделов
- [x] Раздел 2 содержит несколько подразделов
- [x] Заголовки без точки в конце

## 5. Иллюстрации, таблицы, формулы
- [x] Таблицы пронумерованы
- [x] На таблицы есть ссылки в тексте
- [x] Добавлена иллюстрация (архитектурная схема) с подписью и ссылкой
- [x] Формулы оформлены отдельной строкой
- [x] Формулы имеют нумерацию вида (1.1), (2.1)

## 6. Литература и ссылки
- [x] В тексте используются ссылки в квадратных скобках
- [x] Список литературы формируется в порядке упоминания (unsrt)
- [x] Добавлены современные источники (2021+)
- [x] Добавлены официальные документации технологий

## 7. Приложения
- [x] Каждое приложение начинается с новой страницы
- [x] Заголовок "Приложение N" вынесен в верхнюю часть страницы
- [x] В приложениях размещены вспомогательные материалы
- [x] В тексте даны явные ссылки на все приложения

## 8. Объем
- [x] Основной текст содержит ~6950 слов (оценочно соответствует ~30 страницам при 14 pt и 1.5 интервале; уточнить в Overleaf PDF)

## 9. Техническая готовность
- [ ] Проект компилируется локально (XeLaTeX/BibTeX) -- локальные TeX-инструменты отсутствуют
- [x] Подготовлен в Overleaf-совместимой структуре; требуется выбрать компилятор XeLaTeX в настройках проекта
