# Course Report (Overleaf / LaTeX)

## Состав
- `main.tex` - главный файл проекта
- `sections/` - разделы отчета и приложения
- `sections/references.tex` - встроенная библиография (`thebibliography`, без BibTeX)
- `references.bib` - резервный файл библиографии (необязателен для текущей сборки)
- `checklists/methodology_compliance.md` - чек-лист соответствия методичке

## Как собрать в Overleaf
1. Создать новый проект Overleaf и загрузить все файлы из этой папки.
2. В настройках проекта выбрать компилятор `pdfLaTeX` (самый быстрый и стабильный для free-plan).
3. Если нужен именно `XeLaTeX`, текущий шаблон также поддерживает его как fallback.
4. Нажать `Recompile` 2 раза (на втором проходе стабилизируются ссылки и оглавление).

## Что заполнить перед финальной сдачей
- поля на титульном листе (ФИО, группа, дисциплина, руководитель);
- при необходимости скорректировать тему в титуле под официальный формулировочный лист;
- проверить итоговый PDF на предмет точного объема (целевое значение ~30 страниц основного текста).
