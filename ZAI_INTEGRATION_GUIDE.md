## 🔑 API Ключ

**⚠️ ВАЖНО: API ключ не должен быть захардкожен в коде!**

### Как получить Z.AI API ключ:

1. Зарегистрируйтесь на [Z.AI платформе](https://ai.zhipuai.cn/)
2. Перейдите в **API Settings**
3. Создайте новый API ключ
4. Скопируйте ключ

### Безопасное использование:

**НЕ добавляйте ключ в код напрямую!**

Используйте один из способов:
1. Переменные окружения: `ZAI_API_KEY=your_key_here`
2. Файл `.env`: `ZAI_API_KEY=your_key_here`
3. Настройки приложения: в разделе AI Settings

### Настройка в коде:

```dart
final apiKey = Platform.environment['ZAI_API_KEY'] ??
               dotenv.env['ZAI_API_KEY'] ??
               '';  // Получить от пользователя
```

**Никогда не коммитьте реальные API ключи в Git!**

## 🔧 Технические детали

### Z.AI GLM API
- **Endpoint:** `https://open.bigmodel.cn/api/paas/v4/chat/completions`
- **Модель:** `glm-4-plus`
- **Аутентификация:** Bearer Token
- **Формат:** OpenAI-совместимый

### Параметры запроса
```json
{
  "model": "glm-4-plus",
  "messages": [...],
  "temperature": 0.8,
  "top_p": 0.95,
  "max_tokens": 2000,
  "stream": false
}
```

### Обработка ошибок
- Fallback к локальным алгоритмам при ошибке API
- Логирование запросов и ответов
- Graceful degradation

## 🆘 Решение проблем

### Ошибка 401 (Unauthorized)
**Причина:** Неверный API ключ
**Решение:**
1. Проверьте API ключ в настройках
2. Убедитесь, что ключ активен
3. Попробуйте пересохранить ключ

### Ошибка 429 (Rate Limit)
**Причина:** Слишком много запросов
**Решение:**
1. Подождите несколько минут
2. Уменьшите частоту запросов
3. Проверьте лимиты Z.AI

### AI не отвечает
**Причина:** Fallback к локальным алгоритмам
**Решение:**
1. Проверьте интернет-соединение
2. Проверьте статус API в Console
3. Переключитесь на локальный AI

### Настройки не сохраняются
**Причина:** Ошибка SharedPreferences
**Решение:**
1. Перезапустите приложение
2. Очистите кэш приложения
3. Проверьте права доступа
